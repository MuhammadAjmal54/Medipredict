## 🔐 Role-Based Access Control

- **Patient**:
  - Register/Login
  - Book and manage appointments

- **Doctor**:
  - View personal appointments
  - Update profile
  - Check earnings

- **Admin**:
  - Manage doctor profiles
  - Monitor all appointments
  - Full system access
